Finds the closest atom (\char`\"{}neighbor\char`\"{}) to a given point in a frame. 

Given a point and a frame (list of atom coordinates), returns the index of the entry in frame with the lowest distance to the point.~\newline
If atom\+\_\+neighbor is not 0, then only neighbors with it as atomic number may be returned. If 0, then all atoms are eligible.~\newline
Distance is determined using \mbox{\hyperlink{chemistry_8h_a943d08970a42acd0c72248fb0c9017f0}{pbc\+\_\+dist}} (see limitations there).
