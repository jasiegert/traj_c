Calculates the intermolecular RDF from a trajectory. 

Calculates the radial-\/distribution function for the two atom types passed. Distances will be binned into the specified number of bins between min and max distance. Atoms will be assigned to molecules based on their closest atom of specified central atom type. Distances between atoms within the same molecule will be discarded.
