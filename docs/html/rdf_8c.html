Calculates the intermolecular RDF from a trajectory.

Calculates the radial-distribution function for the two atom types passed. Distances will be binned into the specified number of bins between min and max distance. Atoms will be assigned to molecules based on their closest atom of specified central atom type. Distances between atoms within the same molecule will be discarded.

Parameters
[in]frame_nonumber of frames in trajectory.
[in]atom_nonumber of atoms in trajectory.
[in]trajatom coordinates of the trajectory.
[in]pbcperiodic boundary conditions as a 3x3 float array
[in]atomatomic numbers of all atoms in the trajectory.
[in]atom_1first atom type of the RDF pair.
[in]atom_2second atom type of the RDF pair.
[in]atom_centralcentral atom type used to assign atoms to molecules.
[in]min_distancesmallest distance to consider.
[in]max_distancelargest distance to consider.
[in]binnumber of bins and thus number of output values.
[out]rdfsampled distances (average of each bin) and corresponding RDF.
[out]outputoutput string to be passed into stdout and output file.
